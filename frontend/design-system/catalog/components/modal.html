Small Modal sm · max-w-sm

A small confirmation dialog.

Compact modal for quick confirmations or single-field inputs.

Medium Modal md · max-w-md

Default modal size for most dialogs.

Standard modal with description, body content, and footer actions. This is the default maxWidth value used when no size is specified.

Large Modal lg · max-w-lg

Larger modal for forms or detail views.

Used for more complex content such as multi-field forms, session detail previews, or rich text editors where extra horizontal space improves readability.

Extra Large Modal xl · max-w-[80vw]

Full-width modal for code viewers, diffs, or data tables.

XL uses max-w-[80vw] — 80% of the viewport width. Suited for the session conversation view, file diff, or any content that benefits from near-full-screen width.
